# Portfolio de Djerareoutar Nestor

Bienvenue sur mon portfolio en ligne.
Ce repository contient mon profil et mon CV, avec des liens interactifs et un style moderne.

## Contenu du projet

* `profil.html` : Page de profil avec informations personnelles, domaine professionnel, cours en ligne et modal CV.
* `cv.html` : Mon CV complet avec parcours académique, expériences, compétences et centres d’intérêt.
* `style.css` : Fichier CSS commun pour styliser le profil et le CV.
* `upload/nesto.jpg` : Image de profil utilisée sur les pages.

## Fonctionnalités

* Thèmes modifiables (Clair, Ombre, Bleu, Rouge, Vert) via des boutons.
* Aperçu du CV directement depuis le profil via un modal.
* Message informatif pour les cours en ligne (PHP, Java, JavaScript) indiquant qu’ils ne sont pas encore disponibles.
* Responsive design pour mobile et tablette.

## Aperçu

1. Ouvrir `profil.html` pour voir le profil interactif.
2. Cliquer sur **"Aperçu de mon CV"** pour afficher le CV dans une fenêtre modal.
3. Cliquer sur les liens des cours pour voir le message d’information.
